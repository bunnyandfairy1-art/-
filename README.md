# 图片压缩鸭 🦆

macOS 原生图片压缩工具，零依赖，开箱即用。

## 功能特性

- 支持 JPG / PNG / GIF / WebP 格式
- 四档压缩质量：轻度 / 推荐 / 深度 / 极限
- GIF 动画智能抽帧保留
- 文件夹批量压缩，保留目录结构
- 拖拽上传，一键保存
- 内置反馈功能
- 自动检查更新

## 使用方式

1. 从 [Releases](https://github.com/bunnyandfairy1-art/-/releases) 下载最新版 `图片压缩鸭.zip`
2. 解压后双击 `图片压缩鸭.app` 即可使用

## 系统要求

- macOS 10.15+
- Google Chrome 浏览器

## 技术栈

- 纯 Python 标准库 + macOS 原生 sips 命令
- 无需安装任何第三方依赖
